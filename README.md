# 🖥️ Linux Health Monitoring Tool

📌 Description

A lightweight Bash-based tool to monitor system health by tracking CPU, memory, and disk usage. It displays real-time data and logs system performance for basic analysis.

⚙️ How It Works

The main script (monitor.sh) collects system metrics by calling separate modules for CPU, memory, and disk, and stores the results in a log file.

🚀 Features
- Real-time system monitoring
- Modular script design
- Logging of system metrics
- Simple and efficient execution
